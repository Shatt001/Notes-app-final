# Description

Notes tracker app. App allows to create, edit, delete notes. App also able to filter and sort notes by some criteria and saves changes history. <br />
<br/>
App contains 4 pages:<br/>
Home page (List all not deleted notes)<br/>
Add note (add new note)<br/>
Edit note (edit or remove note)<br/>
Notes History (review all changes of notes executed via edit page)<br/>

# Available Scripts

In the project directory, you can run:

### `npm run dev-server`

Runs the app in the development mode.<br />
Open [http://localhost:8082](http://localhost:8082) to view it in the browser.<br/>
The page will reload if you make edits.<br />

### `npm test`
Unit test is not implemented yet.<br/>

### `npm run build`

Builds the app for production to the `build` folder.<br />
It correctly bundles React in production mode and optimizes the build for the best performance.
